                                                         BMI CALCULATOR
A simple and responsive BMI (Body Mass Index) Calculator built using React, React Router, and Vite.
This application allows users to calculate their BMI using both Metric and Imperial units and view their health category with a visual result.

Features

--> BMI calculation using Metric (kg, cm)

--> BMI calculation using Imperial (lb, inches)

--> Automatic unit conversion

--> BMI category detection:

  Underweight               

  Normal

  Overweight

  Obese

--> Result page with image based on category

--> React Router navigation

--> Clean component-based structure

Technologies Used

React

React Router DOM

Vite

JavaScript (ES6+)

CSS
